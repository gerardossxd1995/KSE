Getting Started with StateChart Editor Create a StateChart From Scratch



You are using the StateChart Application, this application gives you the opportunity

to create a StateChart graphically from scratch or by writing a liveness formula and

use the generated model. At first you have to choose the workspace folder, if you press

“cancel” the application will close.



Let's start working a new StateChart by doing one step at the time.

At first you have to choose the way you want to create the Statechart Model.



If you want to write the liveness formula follow the following steps

First Step → Describe the StateChart model by writing a Liveness Formula

Second Step → Initialize and Edit the Graphical representation of the generated model

Third Step → Generate C++ Code For Monas Architecture



If you do not want to write the liveness formula follow these steps

First Step → Create an Empty Diagram File

Second Step → Edit the Empty Diagram File

Third Step → Generate C++ Code For Monas Architecture
