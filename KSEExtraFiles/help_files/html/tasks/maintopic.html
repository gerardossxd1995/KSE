Tasks that you can do with StateChart Editor

You have the capability to edit the diagram with numerous tasks

  • Create new elements, by selecting the element you want to add from the palette and then select the point on the diagram that you want to put the new element.

  • Create an empty diagram, by going to File → New → StateChart Diagram.

  • Create a new liveness formula, by going to File → New → StateChart Formula. Read the instructions for writing a formula.

  • Initialize a diagram for an existing .stct model, by going to File → Initialize kse diagram file

  • Open an existing diagram , by going to File → Open.. or Open URI.

  • Open an existing formula, by going to File → Open Formula. BE CAREFUL if you click on OK the existing model will be overwritten. Read the instructions for writing a formula.

  • Edit the elements, by editing the elements' attributes from properties view or by clicking on the visible attributes of the elements and pressing F2.

  • Put automatically the nodes' label, name and transitions' name, by going to Edit → Labeling Diagram or clicking

    from the toolbar.

  • Edit the activity of a BASIC state, by selecting the BASIC state you would like to edit and going to Edit → Edit Activity or clicking

    from the tool bar.

  • Connect the opened model diagram to a source folder, by going to Edit → Connect Model to source folder or clicking

    from the toolbar.

  • Generate code C++ for the opened model, by going to File → Generate Code for model or clicking

    from the toolbar.

  • Zoom in or out, by going to Diagram → zoom or pressing Ctr+“+” for zoom in and Ctr+“-” for zoom out.

  • Add rulers, page breakers or grid, by going to Diagram → view.

  • Change a diagram element's view, by clicking on it and going to Diagram and choose the characteristic you would like to change, you can change font,line color and line style.

  • Open a new window for the application, by going to Window → Open in new Window.

  • Check if the StateChart is valid according to ASEME and transition's expression grammar. You can find the rules here and grammar here.

  • Save diagram as image, by right clicking on the top node of the part of the diagram you want to save as image and choosing from the pop-up menu File → save as image File....NOTE:The pdf format is unavailable, but you can save it as png, jpeg, jpg, svg, gif and bmp.

  • Print the diagram, by clicking on the print button of the toolbar.
